# Consulta FIPE - Java + Spring Boot

## Sobre o projeto

Aplicação desenvolvida em Java para consulta de veículos utilizando a API pública da Tabela FIPE.

O sistema permite que o usuário consulte carros, motos e caminhões através do terminal, navegando por marcas, modelos e anos disponíveis, retornando os valores atualizados de mercado para cada veículo.

Este projeto foi desenvolvido durante os estudos de Java e Spring Boot, com foco em consumo de APIs REST, manipulação de JSON e utilização de recursos modernos da linguagem Java.

---

## Funcionalidades

* Consulta de carros, motos e caminhões
* Listagem de marcas disponíveis
* Busca de modelos por marca
* Filtro de modelos por trecho do nome
* Consulta de todos os anos disponíveis para um modelo
* Exibição dos valores da Tabela FIPE
* Consumo de API REST
* Conversão de JSON para objetos Java

## Fluxo da aplicação

1. Escolha do tipo de veículo
2. Seleção da marca
3. Busca dos modelos disponíveis
4. Filtro do modelo desejado
5. Consulta dos anos disponíveis
6. Exibição das avaliações FIPE para todos os anos encontrados
